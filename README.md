# edutech_task3
# Task 3 - Exploratory Data Analysis (EDA)

## Dataset
Iris Dataset - 150 flowers, 3 species, 4 measurements

## What I did
- Univariate analysis: histograms and boxplots for each column
- Bivariate analysis: scatter plots and bar charts
- Generated correlation matrix and heatmap
- Created pairplot for multivariate analysis

## Key Insights
- Petal length and petal width are strongly correlated (~0.96)
- Setosa species is clearly separated from the other two
- Virginica has the largest petals on average
- Sepal width has the weakest correlation with other columns
